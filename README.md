# Space_Shooting_Game
우주 슈팅 게임


## 🖥️ 프로젝트 소개
우주에서 날아오는 운석을 파괴하는 슈팅게임입니다.

### ⚙️ 개발환경&기술스택
- Unity, VsCode
- C#
  

## 📌 주요 기능
#### 비행기
- 비행기의 스피드, 상하좌우 움직임
- 미사일과 인터랙선을 위해 우주선에서 미사일 발사 액팅(스페이스바) 시간을 체크
- Raycast를 사용하며 충돌된 물체의 정보를 가지고 옴


#### 미사일
- 비행기에서 발사되도록 함
- target 즉, 운석에 맞으면 메세지를 출력
- 시간이 지난 후 미사일 제거


#### 운석
- 랜덤으로 운석이 움직이게 함
- 게임 map 벗어나면 제거


#### 게임화면 캠
- 게임화면에서 마우스로 물체 시점을 조작
